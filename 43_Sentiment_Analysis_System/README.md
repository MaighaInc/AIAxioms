Sentiment Analysis System
========================

This project demonstrates a complete
sentiment analysis pipeline.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python sentiment_analysis_system.py

Expected Output:
----------------
- Sentiment prediction (Positive / Negative)

Learning Objective:
-------------------
Understand how NLP features
are combined into a real system.
