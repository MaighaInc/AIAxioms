Image Class reopenClassifier using MLP
=====================================

This project demonstrates how a neural network
can classify images using a multilayer perceptron.

Dataset:
--------
Handwritten digits (8x8 images)

Requirements:
------------
- Python 3.x
- numpy
- scikit-learn

Install dependencies:
---------------------
pip install numpy scikit-learn

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python image_classifier_mlp.py

Expected Output:
----------------
- Training loss decreasing
- Test accuracy around 90%+

Learning Objective:
-------------------
Understand how neural networks
process images as numerical vectors.
