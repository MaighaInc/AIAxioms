GRU Time-Series Predictor
========================

This project demonstrates how GRU networks
predict sequential numeric data.

Requirements:
------------
- Python 3.x
- tensorflow
- numpy

Install dependencies:
---------------------
pip install tensorflow numpy

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python gru_time_series_predictor.py

Expected Output:
----------------
- Training loss decreasing
- Correct next-value prediction

Learning Objective:
-------------------
Understand how GRUs handle sequence memory
more efficiently than RNNs and LSTMs.
