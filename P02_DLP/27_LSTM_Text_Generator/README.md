LSTM Text Generator
==================

This project demonstrates how LSTM networks
can generate text character by character.

Requirements:
------------
- Python 3.x
- tensorflow
- numpy

Install dependencies:
---------------------
pip install tensorflow numpy

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python lstm_text_generator.py

Expected Output:
----------------
- Model generates new text
  similar to training data

Learning Objective:
-------------------
Understand how LSTMs learn
language patterns and generate text.
