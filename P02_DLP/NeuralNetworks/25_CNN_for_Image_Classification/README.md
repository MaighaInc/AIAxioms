CNN for Image Classification
============================

This project demonstrates real-world image
classification using Convolutional Neural Networks.

Dataset:
--------
CIFAR-10 (airplane vs automobile)

Requirements:
------------
- Python 3.x
- tensorflow

Install TensorFlow:
-------------------
pip install tensorflow

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python cnn_image_classification.py

Expected Output:
----------------
- Training accuracy improving
- Test accuracy around 85%+

Learning Objective:
-------------------
Understand how CNNs scale
from digits to real images.
