CNN for Digit Recognition
========================

This project demonstrates how Convolutional
Neural Networks classify images efficiently.

Dataset:
--------
MNIST handwritten digits (28x28)

Requirements:
------------
- Python 3.x
- tensorflow

Install TensorFlow:
-------------------
pip install tensorflow

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python cnn_digit_recognition.py

Expected Output:
----------------
- Training accuracy improving
- Test accuracy around 98%+

Learning Objective:
-------------------
Understand how CNNs extract
spatial features from images.
