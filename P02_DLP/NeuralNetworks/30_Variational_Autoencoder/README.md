Variational Autoencoder (VAE)
=============================

This project demonstrates how a Variational
Autoencoder learns a probabilistic latent space.

Requirements:
------------
- Python 3.x
- tensorflow
- numpy

Install dependencies:
---------------------
pip install tensorflow numpy

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python variational_autoencoder.py

Expected Output:
----------------
- Training loss decreasing
- New generated data samples

Learning Objective:
-------------------
Understand probabilistic latent variables
and generative modeling.
