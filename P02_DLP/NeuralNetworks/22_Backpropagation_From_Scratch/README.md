Backpropagation From Scratch
============================

This project demonstrates how neural networks
learn using backpropagation.

Requirements:
------------
- Python 3.x
- numpy

Install numpy:
--------------
pip install numpy

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python backpropagation_from_scratch.py

Expected Output:
----------------
- Loss decreasing over epochs
- Correct XOR predictions

Learning Objective:
-------------------
Understand how gradients flow backward
and how weights are updated.
