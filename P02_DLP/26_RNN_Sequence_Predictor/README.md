RNN Sequence Predictor
=====================

This project demonstrates how Recurrent Neural Networks
learn from sequential data.

Requirements:
------------
- Python 3.x
- tensorflow
- numpy

Install dependencies:
---------------------
pip install tensorflow numpy

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python rnn_sequence_predictor.py

Expected Output:
----------------
- Training loss decreasing
- Correct next-number prediction

Learning Objective:
-------------------
Understand how RNNs remember
previous time steps.
