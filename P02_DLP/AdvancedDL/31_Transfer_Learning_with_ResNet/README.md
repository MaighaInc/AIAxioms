Transfer Learning with ResNet
=============================

This project demonstrates how to use
pretrained deep learning models
to solve new problems efficiently.

Dataset:
--------
CIFAR-10 (airplane vs automobile)

Requirements:
------------
- Python 3.x
- tensorflow

Install TensorFlow:
-------------------
pip install tensorflow

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python transfer_learning_resnet.py

Expected Output:
----------------
- Fast training
- High accuracy with few epochs

Learning Objective:
-------------------
Understand transfer learning
and feature reuse in deep learning.
