Fine-Tuning BERT
================

This project demonstrates how to fine-tune
a pretrained BERT model for text classification.

Task:
-----
Sentiment Analysis (Positive / Negative)

Requirements:
------------
- Python 3.x
- torch
- transformers

Install dependencies:
---------------------
pip install torch transformers

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python fine_tuning_bert.py

Expected Output:
----------------
- BERT training logs
- Correct sentiment prediction

Learning Objective:
-------------------
Understand transformer-based NLP
and fine-tuning pretrained models.
