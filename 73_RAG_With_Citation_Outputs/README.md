RAG with Citation Outputs
========================

This project demonstrates how
RAG systems provide answers
with source citations.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python rag_with_citations.py

Expected Output:
----------------
- Answer text
- List of citation sources

Learning Objective:
-------------------
Understand how RAG systems
build trust with cited answers.
