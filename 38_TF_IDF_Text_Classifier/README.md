TF-IDF Text Classifier
=====================

This project demonstrates how TF-IDF
improves text classification by
weighting important words.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python tfidf_text_classifier.py

Expected Output:
----------------
- Prediction of Spam or Ham

Learning Objective:
-------------------
Understand TF-IDF and how it improves
over simple word frequency.
