Vector Embedding Generator
==========================

This project demonstrates how
text is converted into vector embeddings.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python vector_embedding_generator.py

Expected Output:
----------------
- Vocabulary list
- Numeric vector for each document

Learning Objective:
-------------------
Understand how text is represented
numerically for retrieval and search.
