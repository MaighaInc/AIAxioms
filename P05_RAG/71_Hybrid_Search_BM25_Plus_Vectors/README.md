Hybrid Search Engine (BM25 + Vectors)
===================================

This project demonstrates hybrid search
by combining keyword and semantic retrieval.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python hybrid_search_engine.py

Expected Output:
----------------
- Documents ranked by hybrid relevance score

Learning Objective:
-------------------
Understand why hybrid search outperforms
pure keyword or pure vector search.
