Metadata-Aware Retrieval
=======================

This project demonstrates how
document metadata is used to
filter search results.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python metadata_aware_retrieval.py

Expected Output:
----------------
- Only documents matching metadata filters

Learning Objective:
-------------------
Understand how enterprise RAG systems
control access and relevance.
