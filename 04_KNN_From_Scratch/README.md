K-Nearest Neighbors (From Scratch)
=================================

This project demonstrates a distance-based
classification algorithm.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python knn_from_scratch.py

Expected Output:
----------------
Predicted class based on nearest neighbors.

Learning Objective:
-------------------
Understand instance-based learning
and distance-based classification.
