ReAct stands for:

Earlier agents reacted immediately.

Now let us understand the ReAct loop.

Step one:
the agent receives a task.

Step two:
the agent thinks about what to do.

Step three:
the agent performs an action.

Step four:
the agent observes the result.

Step five:
the agent decides the next step.

This loop continues until the task is solved.

Then the agent performs the action.

The result of the action

Finally,
the agent produces the answer.
