Word2Vec Embedding Trainer
=========================

This project demonstrates how Word2Vec
learns word embeddings from scratch.

Model:
------
Skip-Gram

Requirements:
------------
- Python 3.x
- numpy

Install numpy:
--------------
pip install numpy

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python word2vec_from_scratch.py

Expected Output:
----------------
- Vector embeddings for each word

Learning Objective:
-------------------
Understand how words learn meaning
from surrounding context.
