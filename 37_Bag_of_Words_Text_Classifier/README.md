Bag-of-Words Text Classifier
===========================

This project demonstrates how text
can be converted into numerical vectors
using the Bag-of-Words model.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python bag_of_words_classifier.py

Expected Output:
----------------
- Prediction of Spam or Ham

Learning Objective:
-------------------
Understand how word frequency
can be used for text classification.
