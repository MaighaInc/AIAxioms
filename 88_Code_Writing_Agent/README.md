Code-Writing AI Agent
====================

This project demonstrates how
AI agents generate code
from natural language tasks.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python code_writing_agent.py

Expected Output:
----------------
- Agent plan
- Generated Python code

Learning Objective:
-------------------
Understand how AI systems
translate tasks into code.
