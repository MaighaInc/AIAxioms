Tool-Using AI Agent
==================

This project demonstrates how
AI agents use external tools
to solve tasks.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python tool_using_agent.py

Expected Output:
----------------
- Agent selects appropriate tools
- Executes API calls
- Returns results

Learning Objective:
-------------------
Understand how AI agents interact
with external systems.
