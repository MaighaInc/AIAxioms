Web-Browsing AI Agent
====================

This project demonstrates how
AI agents browse web pages
to retrieve information.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python web_browsing_agent.py

Expected Output:
----------------
- Selected URL
- Extracted webpage content

Learning Objective:
-------------------
Understand how agents navigate
and extract web data.
