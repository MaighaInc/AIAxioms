Short-Term Conversational Memory
================================

This project demonstrates how
chatbots remember recent messages
during a conversation.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python short_term_conversational_memory.py

Expected Output:
----------------
- Bot responses that reference
  recent conversation history

Learning Objective:
-------------------
Understand how short-term memory
maintains conversation context.
