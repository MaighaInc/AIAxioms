EPISODIC MEMORY SYSTEM.

Episodic memory stores experiences.

It answers questions like:
What happened?
When did it happen?
What was the context?

This is different from factual memory.

For example:
Fact memory says:
The user likes Python.

Episodic memory says:
Yesterday, the user asked for Python examples.

Now let us understand the structure.

Each memory episode contains:
a description,
a timestamp,
and context.

When a new query arrives,
we search across episodes
using vector similarity.

This allows the AI
to recall relevant past events.

Now let us look at the code.

We store events
with timestamps.

We convert both the query
and events into vectors.

Then we compute similarity
to find the most relevant events.

This approach is used in:
autonomous agents,
task planning systems,
and reflective AI.

Episodic memory allows AI
to reason over past actions.
