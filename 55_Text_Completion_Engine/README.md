Text Completion Engine
=====================

This project demonstrates how
text completion engines work.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python text_completion_engine.py

Expected Output:
----------------
- Prompt extended with generated words

Learning Objective:
-------------------
Understand how LLMs generate
text token by token.
