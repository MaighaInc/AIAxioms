Genetic Algorithm Optimizer
==========================

This project demonstrates how evolutionary
principles can be used to optimize problems.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python genetic_algorithm_from_scratch.py

Expected Output:
----------------
- Best solution per generation
- Final optimal value

Learning Objective:
-------------------
Understand how optimization can occur
without gradients or derivatives.
