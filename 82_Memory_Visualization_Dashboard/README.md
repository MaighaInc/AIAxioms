Memory Visualization Dashboard
==============================

This project demonstrates how
AI memory can be visualized
for inspection and debugging.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python memory_visualization_dashboard.py

Expected Output:
----------------
- Console-based dashboard showing
  memory types, strengths, and timestamps

Learning Objective:
-------------------
Understand how AI memory can be
inspected and debugged visually.
