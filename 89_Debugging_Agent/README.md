Debugging AI Agent
=================

This project demonstrates how
AI agents detect and debug code.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python debugging_agent.py

Expected Output:
----------------
- Error detected
- Explanation provided
- Fix suggested

Learning Objective:
-------------------
Understand how AI assists developers
in debugging code.
