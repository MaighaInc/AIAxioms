Named Entity Recognition
=======================

This project demonstrates how Named Entity
Recognition works using rule-based logic.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python named_entity_recognition.py

Expected Output:
----------------
- PERSON, LOCATION, ORGANIZATION tags

Learning Objective:
-------------------
Understand how entities are extracted
from raw text.
