Agent Evaluation Framework
=========================

This project demonstrates how
AI agents are evaluated across
multiple dimensions.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python agent_evaluation_framework.py

Expected Output:
----------------
- Accuracy score
- Reasoning score
- Safety score
- Final combined score

Learning Objective:
-------------------
Understand how AI agents are
evaluated before deployment.
