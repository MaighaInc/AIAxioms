RAG Chatbot Over Documents
=========================

This project demonstrates a basic
Retrieval-Augmented Generation chatbot.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python rag_chatbot.py

Expected Output:
----------------
- Answer generated from retrieved document chunks

Learning Objective:
-------------------
Understand how retrieval and generation
work together in RAG systems.
