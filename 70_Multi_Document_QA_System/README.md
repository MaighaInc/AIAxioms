Multi-Document QA System
=======================

This project demonstrates how a
RAG system answers questions across
multiple documents.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python multi_document_qa.py

Expected Output:
----------------
- Answer compiled from multiple documents

Learning Objective:
-------------------
Understand how RAG scales to
enterprise-sized document collections.
