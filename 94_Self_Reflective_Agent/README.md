Self-Reflective AI Agent
=======================

This project demonstrates how
AI agents self-evaluate and improve
their own outputs.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python self_reflective_agent.py

Expected Output:
----------------
- Initial answer
- Self critique
- Improved answer

Learning Objective:
-------------------
Understand how reflection improves
AI output quality.
