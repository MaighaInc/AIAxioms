Decision Tree Classifier (From Scratch)
======================================

This project demonstrates how a decision tree makes
decisions using hierarchical if-else rules.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python decision_tree_from_scratch.py

Expected Output:
----------------
The program predicts whether a customer
will buy a product (Yes / No).

Learning Objective:
-------------------
Understand how decision trees represent
machine-learned rules in a tree structure.
