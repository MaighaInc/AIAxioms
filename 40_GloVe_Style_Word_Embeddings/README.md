GloVe-style Word Embeddings
==========================

This project demonstrates how GloVe
learns word embeddings using global
co-occurrence statistics.

Requirements:
------------
- Python 3.x
- numpy

Install numpy:
--------------
pip install numpy

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python glove_embeddings_from_scratch.py

Expected Output:
----------------
- Training loss decreasing
- Vector embeddings for words

Learning Objective:
-------------------
Understand how global word statistics
capture semantic meaning.
