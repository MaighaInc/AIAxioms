Streaming RAG Responses
======================

This project demonstrates how
RAG answers can be streamed
incrementally to users.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python streaming_rag.py

Expected Output:
----------------
- Answer text printed word-by-word

Learning Objective:
-------------------
Understand how real-time streaming
improves RAG chatbot experience.
