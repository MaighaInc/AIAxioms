Enterprise Knowledge Assistant
==============================

This project demonstrates a complete
enterprise-grade RAG assistant.

Features:
---------
- Vector retrieval
- Metadata filtering
- Citation outputs
- Streaming responses

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python enterprise_knowledge_assistant.py

Expected Output:
----------------
- Streamed answer
- Source citations
- Metadata-respected retrieval

Learning Objective:
-------------------
Understand how production RAG
systems are architected.
