Question Answering System
========================

This project demonstrates a basic
question answering system using
text similarity.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python question_answering_system.py

Expected Output:
----------------
- Sentence that best answers the question

Learning Objective:
-------------------
Understand how QA systems locate
answers within documents.
