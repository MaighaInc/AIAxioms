Naive Bayes Text Classifier
==========================

This project demonstrates how Naive Bayes
classifies text using probability.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python naive_bayes_text_classifier.py

Expected Output:
----------------
The program prints whether a message
is classified as Spam or Ham.

Learning Objective:
-------------------
Understand how probability-based AI models
work with text data.
