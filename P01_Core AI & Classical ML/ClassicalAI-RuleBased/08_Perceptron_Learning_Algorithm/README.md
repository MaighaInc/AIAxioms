Perceptron Learning Algorithm
============================

This project demonstrates the first neural
learning algorithm in AI history.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python perceptron_from_scratch.py

Expected Output:
----------------
- Training progress per epoch
- Convergence message
- Final prediction for a test point

Learning Objective:
-------------------
Understand how neural networks learn
through error correction.
