Linear Regression From Scratch
==============================

This project demonstrates how machines learn
numeric relationships using gradient descent.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python linear_regression_from_scratch.py

Expected Output:
----------------
- Training loss reducing over epochs
- Learned weight and bias values
- Prediction for a new input

Learning Objective:
-------------------
Understand how regression models learn
from error using optimization.
