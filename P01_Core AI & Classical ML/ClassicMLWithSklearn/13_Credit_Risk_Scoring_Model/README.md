Credit Risk Scoring Model (Machine Learning)
===========================================

This project demonstrates how machine learning
is used in banking to assess credit risk.

Requirements:
------------
- Python 3.x
- scikit-learn
- numpy

Install dependencies:
---------------------
pip install numpy scikit-learn

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python credit_risk_scoring.py

Expected Output:
----------------
- Accuracy score
- Confusion matrix
- Classification report
- Risk probability for new customer

Learning Objective:
-------------------
Understand how probability-based models
are used in financial decision making.
