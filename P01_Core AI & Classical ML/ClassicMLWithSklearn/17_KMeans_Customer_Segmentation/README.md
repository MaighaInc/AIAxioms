K-Means Customer Segmentation
=============================

This project demonstrates how businesses
group customers using unsupervised learning.

Requirements:
------------
- Python 3.x
- scikit-learn
- numpy

Install dependencies:
---------------------
pip install numpy scikit-learn

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python kmeans_customer_segmentation.py

Expected Output:
----------------
- Cluster assignments
- Cluster centers
- Cluster prediction for new customer

Learning Objective:
-------------------
Understand how unsupervised learning
discovers patterns without labels.
