Random Forest Classifier
=======================

This project demonstrates ensemble learning
using multiple decision trees.

Requirements:
------------
- Python 3.x
- scikit-learn
- numpy

Install dependencies:
---------------------
pip install numpy scikit-learn

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python random_forest_classifier.py

Expected Output:
----------------
- Accuracy score
- Confusion matrix
- Classification report
- Feature importance
- Prediction for new customer

Learning Objective:
-------------------
Understand how ensemble models improve
accuracy and reduce overfitting.
