Model Evaluation Dashboard
=========================

This project demonstrates how to evaluate
and compare machine learning models
using standard performance metrics.

Requirements:
------------
- Python 3.x
- scikit-learn
- numpy

Install dependencies:
---------------------
pip install numpy scikit-learn

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python model_evaluation_dashboard.py

Expected Output:
----------------
- Accuracy
- Precision
- Recall
- F1 Score
for each model

Learning Objective:
-------------------
Understand how to compare models
and choose the best one for production.
