House Price Prediction (Machine Learning)
=========================================

This project demonstrates a complete
machine learning regression workflow
using scikit-learn.

Requirements:
------------
- Python 3.x
- scikit-learn
- numpy

Install dependencies:
---------------------
pip install numpy scikit-learn

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python house_price_prediction.py

Expected Output:
----------------
- House price predictions
- RMSE and R2 score
- Feature importance
- New house price prediction

Learning Objective:
-------------------
Understand how regression models
are applied in real-world ML projects.
