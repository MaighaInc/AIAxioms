Iris Flower Classification (Machine Learning)
=============================================

This project demonstrates multi-class
classification using the famous Iris dataset.

Requirements:
------------
- Python 3.x
- scikit-learn

Install dependencies:
---------------------
pip install scikit-learn

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python iris_classification.py

Expected Output:
----------------
- Accuracy score
- Confusion matrix
- Classification report
- Prediction for a new flower

Learning Objective:
-------------------
Understand multi-class classification
and model evaluation using a benchmark dataset.
