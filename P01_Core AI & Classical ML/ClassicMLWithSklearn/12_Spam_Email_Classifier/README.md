Spam Email Classifier (Machine Learning)
=======================================

This project demonstrates how machine learning
is used to classify emails as spam or not spam.

Requirements:
------------
- Python 3.x
- scikit-learn

Install dependencies:
---------------------
pip install scikit-learn

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python spam_email_classifier.py

Expected Output:
----------------
- Accuracy score
- Confusion matrix
- Classification report
- Prediction for a new email

Learning Objective:
-------------------
Understand text classification
using vectorization and probabilistic models.
