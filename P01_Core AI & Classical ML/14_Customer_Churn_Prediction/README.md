Customer Churn Prediction (Machine Learning)
============================================

This project demonstrates how machine learning
is used to predict customer churn.

Requirements:
------------
- Python 3.x
- scikit-learn
- numpy

Install dependencies:
---------------------
pip install numpy scikit-learn

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python customer_churn_prediction.py

Expected Output:
----------------
- Accuracy score
- Confusion matrix
- Classification report
- Churn probability for new customer

Learning Objective:
-------------------
Understand how ML models help businesses
retain customers and reduce loss.
