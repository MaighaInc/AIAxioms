Autonomous Research Agent
========================

This project demonstrates an AI agent
that performs research autonomously.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python autonomous_research_agent.py

Expected Output:
----------------
- Research plan
- Search steps
- Final consolidated summary

Learning Objective:
-------------------
Understand how agents research
topics without human guidance.
