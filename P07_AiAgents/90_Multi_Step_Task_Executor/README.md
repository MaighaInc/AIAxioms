Multi-Step Task Executor
=======================

This project demonstrates an AI agent
that executes complex tasks
using multiple steps.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python multi_step_task_executor.py

Expected Output:
----------------
- Planned steps
- Execution logs
- Final completion status

Learning Objective:
-------------------
Understand how autonomous agents
handle multi-step workflows.
