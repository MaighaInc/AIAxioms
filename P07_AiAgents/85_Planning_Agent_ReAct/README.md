Planning Agent (ReAct)
=====================

This project demonstrates a
ReAct-style planning agent.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python planning_agent_react.py

Expected Output:
----------------
- Agent thoughts
- Actions taken
- Observations
- Final answer

Learning Objective:
-------------------
Understand how agents reason
before taking actions.
