PDF Document Ingestion Pipeline
===============================

This project demonstrates how PDF
documents are ingested into a
Retrieval-Augmented Generation system.

Requirements:
------------
- Python 3.x
- PyPDF2

Install dependency:
-------------------
pip install PyPDF2

How to Run:
-----------
1. Place a PDF file in this folder
2. Rename it to sample.pdf
3. Run:

   python pdf_ingestion_pipeline.py

Expected Output:
----------------
- Extracted text chunks from the PDF

Learning Objective:
-------------------
Understand how real documents are
prepared for vector search and RAG.
