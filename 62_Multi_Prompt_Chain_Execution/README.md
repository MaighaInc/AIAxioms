Multi-Prompt Chain Execution
============================

This project demonstrates how
multiple prompts can be chained
to perform complex tasks.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python multi_prompt_chain.py

Expected Output:
----------------
- Output of the final chained prompt

Learning Objective:
-------------------
Understand how LLM workflows
are built using prompt chaining.
