POS Tagger from Scratch
======================

This project demonstrates how Part-of-Speech
tagging works using basic rules and statistics.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python pos_tagger_from_scratch.py

Expected Output:
----------------
- Words tagged as NOUN, VERB, PRON, etc.

Learning Objective:
-------------------
Understand grammatical tagging
and how NLP models identify
word roles in sentences.
