Topic Modeling using LDA
=======================

This project demonstrates how Latent
Dirichlet Allocation discovers topics
from documents.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python topic_modeling_lda.py

Expected Output:
----------------
- Words grouped into topics

Learning Objective:
-------------------
Understand how hidden topics
are discovered in text data.
