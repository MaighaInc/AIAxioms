Tokenizer from Scratch
=====================

This project demonstrates how tokenization
works internally without NLP libraries.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python tokenizer_from_scratch.py

Expected Output:
----------------
- Character-level tokens
- Word-level tokens
- Sentence-level tokens

Learning Objective:
-------------------
Understand how raw text is split
into meaningful units for NLP.
