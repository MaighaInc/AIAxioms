Text Summarization
=================

This project demonstrates extractive
text summarization from scratch.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python text_summarization.py

Expected Output:
----------------
- Short summary of the original text

Learning Objective:
-------------------
Understand how important sentences
are selected automatically.
