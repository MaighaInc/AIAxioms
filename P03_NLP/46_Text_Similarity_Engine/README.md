Text Similarity Engine
=====================

This project demonstrates how cosine similarity
is used to compare text documents.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python text_similarity_engine.py

Expected Output:
----------------
- Similarity score between 0 and 1

Learning Objective:
-------------------
Understand how documents are compared
numerically in NLP systems.
