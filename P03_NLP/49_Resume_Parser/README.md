Resume Parser
=============

This project demonstrates how resumes
can be parsed automatically using NLP.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python resume_parser.py

Expected Output:
----------------
- Extracted name
- Email
- Phone number
- Skills list

Learning Objective:
-------------------
Understand how NLP techniques
are combined into real systems.
