Document Clustering
===================

This project demonstrates how documents
can be grouped automatically based on
their content.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python document_clustering.py

Expected Output:
----------------
- Documents grouped into meaningful clusters

Learning Objective:
-------------------
Understand unsupervised document grouping
using text similarity.
