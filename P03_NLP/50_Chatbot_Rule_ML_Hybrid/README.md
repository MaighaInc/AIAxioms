Hybrid Chatbot (Rule + ML)
=========================

This project demonstrates a chatbot
that combines rule-based logic
with similarity-based NLP responses.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python chatbot_rule_ml_hybrid.py

Example Interaction:
--------------------
You: hello
Bot: Hello! How can I help you?

You: what is ai
Bot: AI stands for Artificial Intelligence.

Learning Objective:
-------------------
Understand how early chatbots
combined rules with NLP techniques.
