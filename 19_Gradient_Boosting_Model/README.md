Gradient Boosting Model
======================

This project demonstrates gradient boosting,
a powerful ensemble learning technique.

Requirements:
------------
- Python 3.x
- scikit-learn
- numpy

Install dependencies:
---------------------
pip install numpy scikit-learn

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python gradient_boosting_model.py

Expected Output:
----------------
- Accuracy score
- Confusion matrix
- Classification report
- Feature importance
- Prediction for new data

Learning Objective:
-------------------
Understand how boosting improves models
by learning from previous errors.
