Diffusion Image Generator (Basics)
=================================

This project demonstrates how diffusion
models generate images by removing noise.

Requirements:
------------
- Python 3.x
- numpy

Install numpy:
--------------
pip install numpy

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python diffusion_basics_image_generator.py

Expected Output:
----------------
- Logs showing progressive denoising
- Final image represented as a matrix

Learning Objective:
-------------------
Understand how diffusion models
generate images from noise.
