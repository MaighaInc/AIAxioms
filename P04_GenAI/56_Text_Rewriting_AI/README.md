Text Rewriting AI
================

This project demonstrates how text
can be rewritten automatically.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python text_rewriting_ai.py

Expected Output:
----------------
- Rewritten version of input text

Learning Objective:
-------------------
Understand how rewriting tasks
are controlled forms of generation.
