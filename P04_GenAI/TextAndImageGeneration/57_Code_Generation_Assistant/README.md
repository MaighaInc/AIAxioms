Code Generation Assistant
========================

This project demonstrates how
AI can generate source code
from natural language instructions.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python code_generation_assistant.py

Expected Output:
----------------
- Python functions generated from instructions

Learning Objective:
-------------------
Understand how natural language
is mapped to programming logic.
