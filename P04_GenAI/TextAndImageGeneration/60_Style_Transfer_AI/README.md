Style Transfer AI
=================

This project demonstrates how AI
combines image content and artistic style.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python style_transfer_ai.py

Expected Output:
----------------
- Description of stylized image

Learning Objective:
-------------------
Understand how AI separates content
and style in images.
