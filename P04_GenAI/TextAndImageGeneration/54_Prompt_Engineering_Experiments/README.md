Prompt Engineering Experiments
==============================

This project demonstrates how
changing prompts changes outputs.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python prompt_engineering_experiments.py

Expected Output:
----------------
- Different responses for different prompts

Learning Objective:
-------------------
Understand how prompts control
model behavior.
