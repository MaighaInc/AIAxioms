Mini GPT-Style Language Model
============================

This project demonstrates the core
idea behind GPT-style language models.

Requirements:
------------
- Python 3.x
- tensorflow
- numpy

Install dependencies:
---------------------
pip install tensorflow numpy

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python mini_gpt_language_model.py

Expected Output:
----------------
- Generated text with GPT-like behavior

Learning Objective:
-------------------
Understand decoder-only Transformers
and causal self-attention.
