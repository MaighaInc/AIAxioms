Image Captioning Model
=====================

This project demonstrates how AI
generates text descriptions from images.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python image_captioning_model.py

Expected Output:
----------------
- A caption describing the image

Learning Objective:
-------------------
Understand how vision and language
are combined in AI systems.
