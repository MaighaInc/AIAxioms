Few-Shot Learning Demo
=====================

This project demonstrates how
LLMs learn tasks using examples
inside the prompt.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python few_shot_learning_demo.py

Expected Output:
----------------
- Correct classification based on examples

Learning Objective:
-------------------
Understand how LLMs generalize
from prompt examples.
