OpenAI-Compatible LLM Wrapper
=============================

This project demonstrates how to
wrap LLM APIs behind a clean interface.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python llm_wrapper.py

Expected Output:
----------------
- Simulated LLM response

Learning Objective:
-------------------
Understand how GenAI applications
interact with LLMs via APIs.
