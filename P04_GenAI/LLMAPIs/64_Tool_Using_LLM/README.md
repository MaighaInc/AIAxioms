Tool-Using LLM
==============

This project demonstrates how
language models can use external tools
to solve tasks.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python tool_using_llm.py

Expected Output:
----------------
- Correct math result using calculator tool
- Normal response for other queries

Learning Objective:
-------------------
Understand how LLMs interact
with external tools and APIs.
