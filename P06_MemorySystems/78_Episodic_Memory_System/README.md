Episodic Memory System
=====================

This project demonstrates how
AI systems store and recall
event-based memories.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python episodic_memory_system.py

Expected Output:
----------------
- Retrieved past events relevant
  to the query

Learning Objective:
-------------------
Understand how episodic memory
stores experiences over time.
