User Personalization Memory
===========================

This project demonstrates how
AI systems personalize behavior
based on user preferences.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python user_personalization_memory.py

Expected Output:
----------------
- Responses adapted to user preferences

Learning Objective:
-------------------
Understand how AI personalizes
interactions using memory.
