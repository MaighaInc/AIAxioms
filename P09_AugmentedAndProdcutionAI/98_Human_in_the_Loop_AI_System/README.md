Human-in-the-Loop AI System
==========================

This project demonstrates how
AI systems incorporate human approval
before taking actions.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python human_in_the_loop_ai.py

Expected Output:
----------------
- AI suggestion displayed
- Human approval requested
- Execution based on approval

Learning Objective:
-------------------
Understand how human oversight
improves AI safety and trust.
