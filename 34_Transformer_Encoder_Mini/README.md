Mini Transformer Encoder
========================

This project demonstrates how a Transformer Encoder
block works internally using NumPy.

Requirements:
------------
- Python 3.x
- numpy

Install numpy:
--------------
pip install numpy

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python transformer_encoder_mini.py

Expected Output:
----------------
- Encoded representations of input tokens

Learning Objective:
-------------------
Understand how attention, feedforward networks,
residual connections, and normalization
work together in Transformers.
