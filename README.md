# AI & Machine Learning Projects Collection

## Setup

### 1. Create Virtual Environment (Recommended)
```bash
cd AI

# Windows(GitBash)
python -m venv .venv
source .venv/Scripts/activate

# Linux/Mac
python3 -m venv .venv
source .venv/bin/activate
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```
