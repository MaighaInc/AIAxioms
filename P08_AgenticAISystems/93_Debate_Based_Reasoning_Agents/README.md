Debate-Based Reasoning Agents
============================

This project demonstrates how
AI agents debate a topic
to reach better conclusions.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python debate_based_reasoning_agents.py

Expected Output:
----------------
- Arguments from multiple agents
- Final judgment decision

Learning Objective:
-------------------
Understand how debate improves
AI reasoning quality.
