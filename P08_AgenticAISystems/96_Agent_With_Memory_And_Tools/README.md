Agent with Memory and Tools
==========================

This project demonstrates a
fully capable AI agent combining:
memory, reasoning, and tools.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python agent_with_memory_and_tools.py

Expected Output:
----------------
- Agent recalls past interactions
- Agent uses tools when needed

Learning Objective:
-------------------
Understand how modern AI agents
integrate memory and tools.
