The agent performs a task.

After each run,

If feedback indicates weakness,
the agent updates itself.

Next time,
the agent produces a better response.

This approach is used in:
long-running agents,
autonomous systems,
and reinforcement learning setups.

Auto-improvement is powerful,
but must be controlled carefully.

Uncontrolled learning
can lead to drift.
