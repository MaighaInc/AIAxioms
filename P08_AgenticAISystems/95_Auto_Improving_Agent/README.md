Auto-Improving AI Agent
======================

This project demonstrates how
AI agents improve their behavior
over multiple executions.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python auto_improving_agent.py

Expected Output:
----------------
- Agent responses improve
  across iterations

Learning Objective:
-------------------
Understand how AI systems
self-improve using feedback.
