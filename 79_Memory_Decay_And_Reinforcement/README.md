Memory Decay and Reinforcement
=============================

This project demonstrates how
AI memory changes over time.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python memory_decay_reinforcement.py

Expected Output:
----------------
- Memories with updated strengths
- Weak memories removed

Learning Objective:
-------------------
Understand how AI decides
what information is important.
