Cross-Session Memory Store
=========================

This project demonstrates how
AI memory persists across
multiple program runs.

Requirements:
------------
- Python 3.x

How to Run:
-----------
1. Run the script once
2. Observe memory saved to memory_store.json
3. Run again to see persisted data

Expected Output:
----------------
- Previously stored values
  available after restart

Learning Objective:
-------------------
Understand how AI memory
is persisted across sessions.
