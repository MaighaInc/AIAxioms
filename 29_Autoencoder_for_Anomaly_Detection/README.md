Autoencoder for Anomaly Detection
================================

This project demonstrates how autoencoders
can detect anomalies without labeled data.

Requirements:
------------
- Python 3.x
- tensorflow
- numpy

Install dependencies:
---------------------
pip install tensorflow numpy

How to Run:
-----------
1. Open terminal
2. Navigate to this folder
3. Run:

   python autoencoder_anomaly_detection.py

Expected Output:
----------------
- Higher reconstruction error for anomalies
- Number of anomalies detected

Learning Objective:
-------------------
Understand unsupervised anomaly detection
using reconstruction error.
